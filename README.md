# Crusader VCs

## Installation

```
$ git clone --recursive git@github.com:gerner/cvc.git
$ cd cvc
$ mkdir build
$ cd build
$ cmake -D TESTS=ON ..
```

If you don't use `git clone --recursive`,  you have to manually setup the submodules with:

```
$ git submodule update --init --recursive
```
